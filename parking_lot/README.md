# Automated ticketing system for Parking lot 


###### The code is written using Python3
 

Sample execution formats :
```
sh bin/setup
```
Run Interractively :
```
sh bin/parking_lot
```
Run taking file as Input :
```
sh bin/parking_lot data/file_input.txt
```
Run using python :
```
python main.py

python main.py data/file_input.txt
```

Run Test cases:
```
python test_app.py
```
